# SUDA-BSTASK-campus_market
苏州大学BS课程大作业校园集市

一个功能完善的校园二手交易平台，支持买卖沟通、议价、商品管理等核心功能。

##  快速开始

```bash
# 1. 进入项目目录
cd campusmarket

# 2. 运行部署脚本（推荐）
chmod +x deploy.sh
./deploy.sh

# 或手动部署
pip install -r requirements.txt
python init_data.py
python run.py
```

访问: http://localhost:5000
